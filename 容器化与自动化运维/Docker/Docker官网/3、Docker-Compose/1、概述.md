# 好处
[[Docker重点]]
* 便捷地启动相关服务，可以一次性按顺序启动服务，可以用作快速测试
* 基于yaml
* 可用于所有环境，生产环境也可以使用
* 需要Dockerfile+docker-compose.yml+compose命令

# 生命周期
* 启动、停止、重建服务
* 查看服务的状态
* 观察服务的启动日志
* 服务上运行一次性命令

# 格式
```yaml
version: "3.9"  # optional since v1.27.0services:
  web:
    build: .
    ports:
      - "5000:5000"
    volumes:
      - .:/code
      - logvolume01:/var/log
    links:
      - redis
  redis:
    image: redisvolumes:
  logvolume01: {}
```
